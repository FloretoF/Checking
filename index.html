Команды Git
Команда Описание
git init Создает новое место, в котором хранится весь код и вся история его изменеий.
git clone url Используется для создания локальной копии удаленного репозитория.
git add file Добавляет указанный файл во временную зону хранения, в котором отслеживаются изменения файлов.
git add . Добавляет все отслеживаемые файлы в текущем каталоге в области промежуточной фиксации.
git commit -m "comment" Используется для фиксации изменений, отслеживаемых в области промежуточной фиксации в локальном репозитории с указанным сообщением.
git commit -a -m "comment" Используется для фиксации всех изменений одним сообщением.
git push url Используется для отправки локальных изменений в удаленный репозиторий по указанному URL-адресу.
git status Используется для отображения состояния рабочего дерева и области промежуточной фиксации. Она предоставляет краткой обзор измнений.
git log Используется для отображения истории фиксаций в репозитории с указанием автора, даты и краткого сообщения фиксации.
git branch branch_name С ее помощью можно добавлять новые ветки, перечислять и переименовывать существующие и удалять их.
git checkout branch_name Позволяет перемещаться между ветками, созданными командой git branch.
git checkout -b branch_name Используется для создания перехода к новой ветке с одним действием. Она создает новую ветку на основе указанного коммита/ветки и переключает рабочую область на новую ветку.
git remote Bспользуется, чтобы указать, на каких удаленных конечных точках будут работать команды синхронизации.
